# readme - Vote Counter (計票器) 使用說明

## 1. 操作步驟

1. 主題修改：直接點擊正上方主題修改文本即可。標題會同步至網頁標題。
2. 更改單次投票數量：直接更改顯示票數或按加號、減號。
3. 投票：點擊左側投票區對應的人即可，每點一次加“單次投票數量”票。
4. 動畫排名：在右側會有實時動畫排名。
5. 設置：**初始化：把所需名單按照英文逗號分隔輸入“人員名單”，然後點擊“確定並保存”。**在右上角點擊設置按鈕即可進入設置界面。你可以編輯相關信息。“主分割線”“次分割線”將排名分為三段。次分割線排名以內的在動畫區將被設為金色；主分割線以內、次分割線以外的則是黃綠色；主分割線以外是淡黃色。
6. 人員編輯：你可以進入人員編輯界面，進行更系統的操作。**直接在設置中的人員名單編輯會導致票數清零，建議單獨改名之類的到這裡操作。**同時，還有按排名批量刪除等。插入功能尚不完善，請在使用後刷新界面。
7. 重置進度：**重置投票。**每次投票完畢後點擊即可清零票數。
8. 查看本地數據：
- 記錄摘要：如果投票中途有事**暫停**，為了**防止數據被篡改**，建議將這一段保存至安全的地方（或發至聊天、社交平臺等保留）。
- 詳細存檔配置：如果你常用這一套人員名單，建議**將這份字符串妥善保留至本地**。否則多種操作都可能會破壞你的備份。
9. 導入到本地數據：如果你要進行數據遷移或因意外需要重新導入備份，就將原先生成的字符串導入到此處。
10. 注意：所有涉及人員名單的分隔符請用**英文逗號**而不是中文逗號。
